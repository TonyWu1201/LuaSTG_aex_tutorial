# LuaSTG aex+ 游戏框架教程
## 关于本仓库
本仓库的目的是为 LuaSTG 教程编写者提供一个可以协作、讨论的空间，并且可以推进 LuaSTG 教程的更新。
### 文件结构
+ **assets**：按照章节存放图像资源
+ **content**：按照章节存放教程正文
+ **unclassified**：未分类的教程
+ **draft**：处于草稿阶段或有待复核的教程
### 如何参与贡献
可以通过以下方式参与教程的修订或提出建议：
1. **提交issue**
2. **创建pull request**
3. **参与讨论**
